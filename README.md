# MIDA — Meeting Intelligence Desktop Agent

AI-агент, который:
- записывает встречи (Zoom, Meet и др.)
- делает транскрипции с разделением по говорящим
- генерирует summary и задачи
- позволяет задавать вопросы по истории встреч
